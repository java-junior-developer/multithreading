1. переключиться на ветку master
git checkout master

2. git pull

3. переключиться на свою ветку
git checkout имя-ветки

4. выполнить слияние с master
git merge master

5. решить вручную возникший конфликт

6. выполнить коммит
(перед выполнением коммита необходимо убедиться, что код
корректно работает)

при необходимости проверить, на какой ветке
находимся: git branch


1. выполнить коммит в своей ветке
2. из своей ветки создать новую - task23:  git branch task23
3. переключиться на task23:  git checkout task23
4. выполнить задание, выполнить коммит
5. переключиться на свою ветку: git checkout имя-ветки
6. слить в нее изменения из ветки task23: git merge task23
